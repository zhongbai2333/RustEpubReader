# RustEpubReader

一个基于 Rust 的 EPUB 阅读器，目前只保留 **Windows 64 位（x86_64）** 支持，并针对“隐蔽阅读 / 办公摸鱼”场景做了较多调整。

## 这次版本的重点

这个分支不再追求多端兼容，核心目标很明确：

- 在 Windows 桌面上提供一个更隐蔽、更轻量的阅读器
- 去掉传统标题栏和任务栏暴露
- 让阅读界面更像一层可控的阅读浮层，而不是一个普通应用窗口

## 隐蔽阅读特性

- **无标题栏窗口**  
  主窗口移除了 Title Bar，视觉上更干净，也更不容易一眼看出是阅读器。

- **任务栏不显示图标**  
  打开阅读器后不会在任务栏单独挂一个常规应用入口。

- **Boss Key / 摸鱼快捷键**  
  支持全局隐藏 / 显示阅读器，用于快速收起窗口。

- **快捷键可捕捉修改**  
  不需要手动输入字符串，点击修改后直接按键即可设置。

- **支持单键和组合键**  
  默认是 `F1`，也支持组合快捷键。  
  为了尽量减少与其他软件冲突，单键模式限制为 `F1-F12`。

- **快捷键冲突检测**  
  Windows 下通过系统热键注册机制处理，如果快捷键已被其他程序占用，会直接提示冲突。

- **阅读界面默认收起 Chrome**  
  工具栏、目录、右上页码、右下章节数默认隐藏。  
  按 `F2` 可以统一显示 / 隐藏这一整组阅读界面元素。

- **阅读窗口始终置顶**  
  进入阅读页后窗口会保持在最上层，避免点击其他软件后阅读器被压到后面。

- **任意空白区域拖动窗口**  
  由于没有标题栏，现在可以在阅读器的空白区域直接拖动窗口位置。

- **四周边缘可调整大小**  
  鼠标悬停到窗口边缘或四角时，可以像普通窗口一样拖拽缩放。

- **背景透明度更适合摸鱼场景**  
  调整的是阅读背景层，而不是把整窗连同文字一起做淡化，目标是让文字保持可读、背景自然融入桌面。

- **滚动模式隐藏滚动条**  
  阅读模式为滚动时，滚动条默认不显示，但滚轮 / 触控板滚动仍然可用。

## 常用快捷键

- `F1`  
  全局隐藏 / 显示阅读器（默认，可在设置中修改）

- `F2`  
  显示 / 隐藏阅读界面元素：
  - 工具栏
  - 目录
  - 页码
  - 章节信息

- `A / D`  
  上一章 / 下一章

- `← / →`  
  翻页或切换阅读位置（保留原有阅读操作）

## 阅读能力

除了隐蔽性调整，这个版本仍保留常规阅读能力：

- EPUB 打开与书库记录
- 阅读进度保存
- 滚动 / 翻页两种模式
- 背景色、背景图、字体、字号、字体颜色调整
- 中文 / English 界面

## 项目结构

```text
RustEpubReader/
├── core/      # EPUB 解析、书库、i18n、共享等核心逻辑
└── desktop/   # Windows 桌面端（egui / eframe）
```

## 构建

本项目当前面向 Windows 64 位：

```bash
cargo build --release -p rust_epub_reader
```

如果你本地缺少完整的 Windows MSVC 工具链，也可以直接使用 GitHub Actions 进行编译。

## GitHub Actions 构建产物

仓库已经配置了 Windows 构建工作流。  
推送代码后，GitHub Actions 会自动生成：

- `RustEpubReader-Win64-<branch>.exe`

你可以在对应 workflow run 的 artifact 中下载编译结果。

## 当前定位

这不是一个追求“全平台统一体验”的阅读器分支。  
它现在更像是一个专门为 Windows 办公环境优化的隐蔽 EPUB 阅读器。

如果你的目标是：

- 低暴露
- 快速隐藏
- 阅读界面尽量干净
- 在桌面环境中更自然地悬浮阅读

这个分支就是按这个方向做的。
